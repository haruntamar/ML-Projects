## Projects

### 1. EA FC 24 Player Value Predictor
**Directory**: `eafc24-value-predictor/`

Predicts player market values in EA Sports FC 24 using Random Forest Regression based on player statistics.

- **Model**: Random Forest Regressor
- **Performance**: R² Score of 0.9665
- **Key Features**: Player attributes (pace, shooting, passing, defending, dribbling, physicality)
